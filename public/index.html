Bible Plan RSS Feed Generator

Simple to use RSS Feed Generator for reading a specified number of chapters per day from a feed reader.

How to use

Build an RSS feed url like the below examples. You specify the plan using the plan code (ot,nt,full), the translation code, start date, and number of chapters you want to have added to to your RSS Feed each day. Note, the start day should be the date for yesterday for day 1.

Examples (note all will show the entire plan due to date being so far in past and should not be used)

Notes:Dates are in yyyyMMdd format so June 5th, 2023 = 20230605
